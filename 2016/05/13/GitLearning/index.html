GitLearning

  • 学习


由于最近要和朋友协同开发一个项目，所以复习了一遍Git。决定写一篇博客来记录一下温习的成果。看完第一句是不是把我脑补成编程大神了，哈哈，你上当了！其实我是编程小白。额。。好吧，开始记录我的复习成果。

###1.常用命令
1.新建一个文件夹 mkdir file
2.初始化git本地仓库 git init
3.显示当前目录下的文件 ls
4.回到上一级目录 cd ..
5.添加到暂存区 git add file
6.提交到当前分支 git commite -m”Comment”
7.版本回退 git reset –hard HEAD^
8.查看仓库当前状态 git status
9.查看不同 git diff
10.撤销修改 git checkout –file
11.删除文件 rm file
12.查看日志 git log
13.查看命令操作日志 git reflog
14.查看文件详情 cat file
15.撤销暂存区修改 git reset HEAD file
16.本地关联远程库 git remotea add origin (copy address)
17.将master分支提交到远程 git push origin master
18.查看当前分支 git branch
19.从远程库克隆 git clone (copy address)
20.创建新分支 git branch BranchName
21.切换分支 git checkout BranchName
22.创建分支并切换 git checkout -b BranchName
23.合并某分支到当前分支 git merge BrancName
24.有记录的合并分支 git merge –no-f -m”” BranchName
25.删除某个分支 git branch -d BranchName

目前先写到这，在使用过程过慢慢补充~
